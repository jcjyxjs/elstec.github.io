记一次学校电脑装Mojave10.14.5

jcjyxjs 2020-05-15 in Mac · Views

前言

学校电脑是政府统一采购的品牌机，本来抱着试一试的想法装，结果直接就成功引导进安装了。

具体型号是： HP ProDesk 480 G4 MT Business PC

7代i3，H110的板子。加一张从来没听过的 R7 430 .(750TI是我之前加的)。我的想法是拔掉独显，用7代的 HD630 驱动。

不过这块板子板载只有 VGA 和 DP ， VGA 肯定不用想了，能点亮但是不能驱动，学校显示器又只有 VGA 和 24+1 。所以我就买了一个 DP 转 VGA 。

过程

我下载的镜像是黑果小兵的10.14.5-18F132 。

镜像制作完成之后，重新进入BIOS设置时间为 2017 或 2015 。

保存重启使用 U盘 启动 。

选择 Boot macOS Install from Install macOS Mojave 。

-v 跑完命令行直接就进了安装界面。

首先还是要抹盘。磁盘工具直接把整个盘抹了，我不是双系统所以不在意。抹盘两种格式都行，不影响。

然后就是下一步下一步，选择安装盘安装。

安装完后会重启。再次选择 U盘 启动。选择 Boot macOS Install from mac 。

然后就进入了安装界面，进行漫长等待，有些机器进入安装之后会重启一次，继续选择 Boot macOS Install from mac 就行了。

跑完之后就进入到了熟悉的设置界面。下一步下一步设置就可以了。

再次重启进入 clover 按 o 选择 config 引导文件。直接使用 HD630 进行引导。进系统之后显卡驱动。

百度随便搜一个 clover 安装，然后挂载系统 EFI 分区和 U盘EFI 分区 。把 U盘EFI 分区的内容全部拷贝到系统 EFI 分区中，把需要的引导文件名改成 config.plist 然后启动系统就不需要 U盘 了。

结束之后，基本能用。但我这品牌机声卡是科胜讯的，网上找不到相关驱动，我也不会做，就买了一个 usb 声卡驱动凑合着用[doge]。
